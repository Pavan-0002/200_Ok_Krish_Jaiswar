# 💰 PocketSense AI

An AI-powered personal finance coach for students.

## 🚀 Features

* Expense tracking
* Savings tracking
* On-screen alerts (real-time)
* Profile management
* Budget & goal tracking

## 🧠 Smart Features

* Alerts for unusual spending
* Savings progress tracking
* Daily financial insights

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Spring Boot
* Database: Supabase (PostgreSQL)
* Deployment: Vercel + Render

## ▶️ How to Run

### Backend

```
mvn spring-boot:run
```

### Frontend

Open `index.html` in browser

## 🌟 Project Goal

To help students manage money smarter using intelligent insights and real-time feedback.
